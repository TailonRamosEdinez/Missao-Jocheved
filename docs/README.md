# Missão Jocheved

Esta pasta é a base inicial de um site religioso, com estrutura simples para crescer em etapas.

## Estrutura criada

- index.html: página inicial
- pages/: páginas internas para mídia, FAQ, newsletter e contato
- assets/css/: estilos
- assets/js/: comportamento dinâmico
- assets/img/: imagens e ilustrações
- data/: conteúdo em formato JSON para evolução futura
